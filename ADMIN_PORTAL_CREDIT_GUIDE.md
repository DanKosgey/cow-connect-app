# Admin Portal Credit User Guide

## Overview
This guide explains how administrators can manage the credit system through the Admin Portal, including farmer credit profiles, request approvals, and system analytics.

## Getting Started
1. Log in to the Admin Portal using your admin credentials
2. Navigate to "Credit Management" from the main menu
3. Access different credit management sections through the tab interface

## Credit Management Dashboard

### Farmer Overview
- View all farmers with credit profiles
- See credit utilization metrics for each farmer
- Filter farmers by credit status (High Credit, Low Credit, No Credit)
- Search for specific farmers by name or phone number

### Pending Requests
- See the number of pending credit requests requiring approval
- Access the approval queue directly from the dashboard
- Prioritize requests based on farmer needs and credit history

### System Analytics
- View key credit metrics (total outstanding, issued this month, active users)
- Access detailed analytics reports
- Monitor system performance and identify trends

## Managing Farmer Credit

### Credit Profile Management
1. Navigate to the "Farmers" tab in Credit Management
2. Select a farmer to view their credit profile
3. View credit tier, limit, balance, and utilization
4. Adjust credit limits if needed
5. Freeze or unfreeze accounts when necessary

### Granting Credit
1. Identify farmers who are eligible for credit but haven't received it
2. Click "Grant Credit" for the selected farmer
3. System automatically calculates eligible amount based on pending collections
4. Confirm the credit grant
5. Farmer receives notification of credit availability

### Freezing/Unfreezing Accounts
1. Locate farmers with account issues
2. Click on the farmer to view details
3. Use the "Freeze/Unfreeze Credit" action
4. Provide a reason for freezing if applicable
5. Confirm the action

## Credit Request Approval

### Approval Queue
1. Navigate to the "Approvals" tab in Credit Management
2. View all pending credit requests
3. Sort requests by submission date or farmer name
4. Click "View Details" to see request specifics

### Approving Requests
1. Review request details including farmer information and product details
2. Verify farmer's credit eligibility and available balance
3. Click "Approve" to authorize the request
4. System updates farmer's credit balance and notifies relevant parties

### Rejecting Requests
1. Review request details
2. Click "Reject" if the request should not be approved
3. Provide a clear reason for rejection
4. System notifies the farmer of the rejection

## Credit Analytics and Reporting

### Analytics Dashboard
1. Access through the "Analytics" tab in Credit Management
2. View key metrics and trends
3. Monitor credit utilization rates
4. Identify farmers at risk of default

### Risk Assessment
1. View risk distribution among farmers
2. Identify high-risk accounts requiring attention
3. Monitor default rates and outstanding amounts
4. Generate reports for management review

### Credit Settings
1. Configure global credit parameters
2. Set default credit percentages by tier
3. Adjust maximum credit limits
4. Configure alert thresholds and notifications

## Troubleshooting

### Common Issues
- **"Unable to Grant Credit"**: Verify farmer eligibility and check for existing credit
- **"Request Approval Failed"**: Check system connectivity and try again
- **"Data Not Updating"**: Refresh the page or check real-time connection status

### System Maintenance
- Regularly review and update credit settings
- Monitor system performance and error logs
- Ensure database backups are functioning properly
- Coordinate with technical team for system updates

## Best Practices
- Regularly review pending requests to avoid delays
- Monitor farmer credit utilization to identify potential issues
- Maintain clear communication with farmers about their credit status
- Document significant credit limit adjustments or account freezes
- Use analytics to identify trends and improve the credit system
