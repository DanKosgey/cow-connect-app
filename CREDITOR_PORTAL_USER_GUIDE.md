# Creditor Portal User Guide

## Overview
This guide explains how agrovet creditors can manage credit requests and track farmer credit activities through the Creditor Portal.

## Getting Started
1. Log in to the Creditor Portal using your agrovet credentials
2. Navigate through the dashboard to access different credit management sections
3. Use the main navigation cards to access specific functionalities

## Creditor Dashboard

### Quick Access Features
- **Credit Management**: Manage farmer credit requests and approvals
- **Credit Reports**: View detailed analytics and reports
- **Farmer Profiles**: Access detailed farmer credit information
- **Payment Tracking**: Monitor credit transactions and repayments

### Quick Stats
- **Pending Applications**: Number of credit requests awaiting action
- **Total Credit Issued**: Overall credit volume in the system
- **Active Farmers**: Number of farmers with active credit profiles

## Credit Management

### Viewing Credit Requests
1. Click on "Credit Management" from the dashboard
2. View all credit requests in a sortable table
3. Filter requests by status (Pending, Approved, Rejected)
4. Search for specific requests by farmer name, phone, or product

### Request Details
1. Click "View" for any request to see detailed information
2. Review farmer profile and credit status
3. Check available credit at time of request
4. View request history and notes

### Managing Requests
- **Pending Requests**: Monitor and prepare for upcoming disbursements
- **Approved Requests**: Process and disburse products to farmers
- **Rejected Requests**: Review reasons and follow up if needed

## Credit Reports and Analytics

### Accessing Reports
1. Click on "Credit Reports" from the dashboard
2. View various analytical charts and metrics
3. Filter reports by time range (7, 30, or 90 days)

### Key Reports
- **Risk Distribution**: See how farmers are distributed across risk levels
- **Credit Utilization**: Monitor overall system credit usage
- **Payment Tracking**: Follow credit repayment patterns
- **Transaction History**: Detailed view of all credit transactions

### Exporting Data
- Export reports in various formats for management review
- Schedule automatic report delivery
- Customize report metrics and filters

## Farmer Profiles

### Viewing Farmer Information
1. Click on "Farmer Profiles" from the dashboard
2. Browse all farmers with credit profiles
3. Filter by credit tier (New, Established, Premium)
4. Filter by account status (Active, Frozen)

### Profile Details
- **Credit Tier**: Farmer's status level affecting credit limits
- **Credit Limit**: Maximum credit allowance
- **Available Credit**: Current usable credit balance
- **Utilization**: Percentage of credit limit used
- **Account Status**: Active or frozen status

### Monitoring Farmer Status
- Track high-utilization farmers who may need attention
- Identify frozen accounts requiring resolution
- Monitor new farmers as they establish credit history

## Payment Tracking

### Transaction Monitoring
1. Click on "Payment Tracking" from the dashboard
2. View all credit transactions in chronological order
3. Filter by transaction type (Granted, Used, Repaid, Adjusted, Settlement)
4. Search for specific transactions by farmer information

### Analytics Charts
- **Payment Volume**: Visualize credit flow over time
- **Net Credit Change**: Track overall system credit balance
- **Transaction Trends**: Identify patterns in credit usage

### Transaction Details
- **Amount**: Value of the transaction
- **Balance Before/After**: Credit balance changes
- **Description**: Details about the transaction
- **Date**: When the transaction occurred

## Troubleshooting

### Common Issues
- **"Unable to View Request"**: Refresh the page or check internet connection
- **"Data Not Loading"**: Wait a moment and try again; system may be updating
- **"Access Denied"**: Verify you have creditor permissions

### Getting Help
- Contact system administrator for access issues
- Report bugs or issues through the support system
- Request feature enhancements through proper channels

## Best Practices
- Regularly check pending requests to ensure timely processing
- Monitor high-risk farmers and coordinate with admin team
- Use analytics to understand credit usage patterns
- Maintain accurate records of all disbursements
- Communicate with farmers about their credit status
- Coordinate with admin team on policy changes or system updates
