# Google Play Store Assets Guide

## Required Assets for Play Store Submission

### 1. App Icon
**Specification:**
- Size: 512 x 512 pixels
- Format: PNG (32-bit)
- No transparency
- Safe zone: 66dp from edges

**Current Status:** ✅ You have `icon.png` (update to 512x512 if needed)

**How to Create:**
```bash
# Using Figma, Canva, or any image editor
# Export your logo as 512x512 PNG
# Make sure it's clear and recognizable
```

---

### 2. Feature Graphic
**Specification:**
- Size: 1024 x 500 pixels
- Format: PNG or JPEG
- No transparency

**What to Include:**
- App name: "Cow Connect Collector"
- Tagline: "Track Milk Collections Effortlessly"
- Simple graphics (milk can, location pin, checkmark)
- Brand colors (Green: #16A34A)

**Template Idea:**
```
[Cow Icon] | COW CONNECT COLLECTOR | Track milk collections on the go
```

---

### 3. Screenshots
**Specification:**
- Minimum: 2 screenshots
- Maximum: 8 screenshots
- Size: 1080 x 2340 pixels (phone) or 2048 x 2732 (tablet)
- Format: PNG or JPEG

**Recommended Screenshots:**
1. **Login Screen** - Show clean authentication
2. **Home Dashboard** - Display summary cards and recent collections
3. **New Collection Form** - Show farmer selection and input fields
4. **Collection History** - Show sync status with badges
5. **Performance Charts** - Display earnings and metrics
6. **Offline Mode** - Show "Offline" indicator

**How to Capture:**
```bash
# On Android device/emulator:
# Take screenshots using device buttons
# Or use Android Studio's screenshot tool
# Resize to 1080 x 2340 if needed
```

---

### 4. Short Description
**Specification:**
- Maximum: 80 characters
- Plain text, no formatting

**Suggested:**
```
Track dairy milk collections offline. Sync collections and manage farmers.
```
(76 characters)

---

### 5. Full Description
**Specification:**
- Maximum: 4000 characters
- Can include formatting and lists

**Template:**
```markdown
Cow Connect Collector - Your Complete Milk Collection Solution

FEATURES:
✅ Offline Collection Recording
Record milk collections even without internet - data syncs automatically when online.

✅ GPS Verification
Every collection is GPS-tagged for authenticity and route tracking.

✅ Photo Documentation
Optional photo capture for quality control and record verification.

✅ Real-Time Sync
Automatic cloud synchronization keeps your data safe and accessible.

✅ Performance Tracking
View your earnings, collection stats, and performance metrics.

✅ Farmer Management
Quick access to farmer profiles and collection history.

PERFECT FOR:
- Dairy milk collectors
- Field agents
- Collection supervisors

SECURE & RELIABLE:
- Encrypted data storage
- Secure authentication
- Automatic backups

Download Cow Connect Collector today and streamline your milk collection operations!
```

---

### 6. App Category
**Select:** Business **or** Productivity

---

### 7. Content Rating
**Questionnaire Answers:**
- Violence: No
- User  Interaction: No
- Shares Location: Yes (explain: "GPS for collection verification")
- Shares Personal Info: No
- Unrestricted Web Access: No
- Paid Features: No

**Expected Rating:** Everyone
